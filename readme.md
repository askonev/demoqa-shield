# demoqa_shield

<https://demoqa.com/>

A test automation project using py playwright.

## Installation

```bash
# For pyproject.toml via uv
uv venv
uv pip compile pyproject.toml -o requirements.txt
uv pip install -r requirements.txt
```

```bash
# For requirements.txt
uv venv
uv pip install -r requirements.txt
```

### playwright

```bash
playwright install
```
